# registration_form_cardstyle

🎓 Student Registration Form (PHP + Bootstrap)
A simple and modern Student Registration Form built using PHP and Bootstrap 5.  </br> This is a beginner-friendly project designed to demonstrate form handling, validation, and responsive UI design in a single-page application.

🚀 Features
✅ One-page PHP form (no page reload complexity) </br>
🎨 Modern UI with Bootstrap (card layout + centered design) </br>
📱 Fully responsive design </br>
🔒 Input sanitization for security </br>
    ✔️ Server-side validation: </br>
        Name (required) </br>
        Email (valid format) </br>
        Phone number (10 digits) </br>
        Age (1–100 range) </br>
        Gender selection </br>
        Qualification (required) </br>
        Course (required) </br>
    ⚠️ Error messages display </br>
    🎉 Success message on valid submission </br>

🛠️ Technologies Used </br>
    PHP (Core PHP) </br>
    HTML5 </br>
    CSS3 </br>
    Bootstrap 5 </br>

📂 Project Structure </br>
/project-folder </br>
│── index.php   # Main file (form + validation + UI) </br>

▶️ How to Run </br>
    Install XAMPP / WAMP / MAMP </br>
Place the project folder inside: </br>
    htdocs (XAMPP) </br>
    Start Apache server </br>
Open browser and visit: </br>
    http://localhost/index.php
