# registration_form_cardstyle

🎓 Student Registration Form (PHP + Bootstrap)

A simple and modern Student Registration Form built using PHP and Bootstrap 5. This is a beginner-friendly project designed to demonstrate form handling, validation, and responsive UI design in a single-page application.

🚀 Features

✅ One-page PHP form (no page reload complexity)

🎨 Modern UI with Bootstrap (card layout + centered design)

📱 Fully responsive design

🔒 Input sanitization for security

✔️ Server-side validation:

Name (required)

Email (valid format)

Phone number (10 digits)

Age (1–100 range)

Gender selection

Qualification (required)

Course (required)

⚠️ Error messages display

🎉 Success message on valid submission

🛠️ Technologies Used

PHP (Core PHP)

HTML5

CSS3

Bootstrap 5

📂 Project Structure
/project-folder
│── index.php   # Main file (form + validation + UI)
▶️ How to Run

Install XAMPP / WAMP / MAMP

Place the project folder inside:

htdocs (XAMPP)

Start Apache server

Open browser and visit:

http://localhost/index.php
